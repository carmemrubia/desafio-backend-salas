# Sistema de Gestão de Alocação de Salas

API REST desenvolvida em Node.js com AdonisJS para gerenciar alocação de salas entre professores e alunos.

##  Tecnologias

- Node.js v18+
- AdonisJS v6
- PostgreSQL
- Lucid ORM

##  Pré-requisitos

- Node.js instalado
- PostgreSQL instalado
- Git instalado
